# Customer Behavior Analysis

## Project Overview
This project analyzes customer transaction data to understand purchasing patterns, customer segmentation, payment preferences, and churn behavior.

The goal is to generate insights that help businesses improve customer engagement and retention.

## Dataset
Customer Behavior Analysis Dataset from Kaggle.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Product Category Sales Analysis
5. Payment Method Distribution
6. Customer Churn Analysis
7. Customer Segmentation (Age Groups)
8. Customer Retention Analysis

## Key Insights
- Some product categories generate significantly higher sales.
- Customers show strong preferences for certain payment methods.
- A percentage of customers churn, indicating retention opportunities.
- Specific age groups contribute more to total revenue.

## Business Recommendations
1. Focus marketing efforts on high-performing product categories.
2. Promote popular payment methods with incentives.
3. Implement loyalty programs to reduce churn.
4. Target high-spending age groups with personalized marketing.
5. Improve engagement through personalized offers and communication.
